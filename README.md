🩺 Pulsifer

Projeto desenvolvido para a faculdade com foco em um relógio inteligente voltado para a área da saúde, utilizando conceitos de IoT.

🚀 Sobre o projeto

O Pulsifer é um conceito de smartwatch capaz de monitorar sinais vitais em tempo real e alertar em situações de risco, trazendo mais segurança principalmente para idosos e pessoas com condições médicas.

💻 Tecnologias utilizadas
HTML5
CSS3
Font Awesome
🎯 Funcionalidades
Layout responsivo
Seção de apresentação do produto
Cards informativos com destaque visual
Interface moderna e limpa
